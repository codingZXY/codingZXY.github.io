概览

之前一直想用Markdown来写东西，苦于找不到一个好用的Markdown编辑器，直到发现了Typora，用上之后就停不下来了。Typora是一个简洁优雅且免费的Markdown编辑器，它支持多平台（Mac/Windows/Linux），下载链接在这里。以下内容参考自Typora的官方手册，原文链接在这里。
